## Requirements

- Go `1.26.4` or newer.
- Git.
- Node is only needed when working on this documentation site.

## Install the CLI

Install the published CLI:

```bash
go install github.com/cybersaksham/gogo/cmd/gogo@v0.2.0
```

Check the installed binary:

```bash
gogo version
```

If a newly published tag is not visible through the public Go checksum database
yet, install directly from Git:

```bash
GOPROXY=direct GONOSUMDB=github.com/cybersaksham/gogo \
  go install github.com/cybersaksham/gogo/cmd/gogo@v0.2.0
```

## Use Gogo as a dependency

Add the framework module to an application:

```bash
go get github.com/cybersaksham/gogo@v0.2.0
```

Generated projects already reference the public module path. Run `go mod tidy`
after changing apps or imports.
